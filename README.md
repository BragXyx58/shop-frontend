Клиентская часть интернет-магазина (React+TypeScript+Vite+Zustand+i18next).

===========НАСТРОЙКА============

Установление зависимостей:
npm install

Запуск сервера разработки:
npm run dev

===============ИСПОЛЬЗОВАНИЕ===========

Открыть http://localhost:5173

Тестовый сценарий:
1. Зарегистрировать первого пользователя (автоматически получает роль Admin)
2. Перейти в "Админ панель" -> создать новый товар
3. Выйти и зарегистрировать обычного клиента.
4. Добавить товар в корзину, заполнить адрес/телефон и оформить заказ.
5. Зайти в Профиль и проверить наличие заказа.
6. Проверить терминал бэкенда на наличие логов от RabbitMQ.
7. Зайти под Админом -> изменить статус заказа
